基础指令组dummy
指令/功能 列表
- spawn 召唤假人
- remove 移除假人
  - 指定假人
  - 全部假人
- reload 重载配置文件
- config 配置假人功能
- skin 修改假人皮肤
- list 查看假人列表
- exp 转移假人经验到玩家
- inv 查看假人背包、装备栏等
- tpto 将玩家传送到假人处
- tphere 将假人传送到当前玩家处
- tps 交换玩家和假人的位置
- actions 通过指令控制假人动作
  - attack
  - chat
  - command
  - drop
  - hold
  - jump
  - look
  - mine
  - mount
  - move
  - sleep
  - sneak
  - sprint
  - swap
  - use
  - wakeup
  - stop 停止当前动作
  - 参数说明
    - 指定动作
    - 指定假人
    - 动作类型(按自定义时间间隔持续、单次)

假人功能
- 加载区块
- 右键假人打开假人容器gui，支持进行假人配置、管理假人背包、装备栏等、管理假人皮肤
- 假人支持的配置
  - 无敌模式
  - 是否启用碰撞箱
  - 幽灵模式(非实体、不加载区块，只保留外观)
